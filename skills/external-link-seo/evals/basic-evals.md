# External Link SEO - Evaluation Tests

## Test Cases

### 1. Link Profile Audit
**Input:** "Analyze my backlink profile and tell me what's working and what's not"
**Expected:** Agent should ask for domain URL, check current metrics (DA, referring domains), and provide audit framework

### 2. Competitor Link Analysis
**Input:** "Where are my competitors getting their backlinks from?"
**Expected:** Agent should identify competitors, analyze their link sources, and recommend similar opportunities

### 3. Link Building Strategy
**Input:** "I need to build 50 quality backlinks in the next 3 months"
**Expected:** Agent should create realistic strategy with timeline, resource requirements, and specific tactics

### 4. Toxic Link Identification
**Input:** "I think I have some spammy backlinks hurting my site"
**Expected:** Agent should guide through link audit, identify toxic patterns, and recommend disavow process

### 5. Outreach Campaign
**Input:** "Help me create a link building outreach campaign"
**Expected:** Agent should provide target identification, email templates, follow-up sequence, and tracking framework

### 6. Guest Post Pitch
**Input:** "Write a guest post pitch for [industry publication]"
**Expected:** Agent should create personalized pitch with topic ideas, credentials, and value proposition

### 7. Broken Link Building
**Input:** "How do I find and capitalize on broken link opportunities?"
**Expected:** Agent should explain process, provide tools, and give outreach templates

### 8. Link Reclamation
**Input:** "I found some sites mentioning my brand without linking"
**Expected:** Agent should provide reclamation email templates and follow-up strategy
